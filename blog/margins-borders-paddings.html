Margins, Borders, and Padding

Summary

Margins, borders, and padding are three important CSS properties that control the space around and inside elements. Understanding how they work helps you create clean, balanced, and well-structured layouts for your webpages.

Margin

The margin is the space outside an element. It creates distance between the element and other elements around it. Think of it as the "outer buffer zone." For example:

div { margin: 20px; }

This will add 20px of space outside all sides of the div, pushing it away from other elements.

Border

The border is the line that goes around an element. It sits between the margin and the padding. Borders can have thickness, style, and color. For example:

div { border: 2px solid black; }

This will give the div a solid black border 2px thick around it.

Padding

The padding is the space inside an element, between its content and the border. Padding increases the distance between text (or other content) and the edge of the box. For example:
